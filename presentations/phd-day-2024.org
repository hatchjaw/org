* Differentiable Programming

- Differentiate a program (analytically) with respect to its inputs
  + Automatic differentiation
- Compute the sensitivity of a program's outputs to changes in its inputs
- Key to gradient-based optimisation problems
- Native support in some languages, e.g. Swift
- Support via libraries in Python, Julia, etc.

* Differentiable DSP

- Application of differentiable programming to audio tasks
  + Source separation
  + Timbre transfer
  + Parameter estimation, etc.
- Possible in Faust via the DawDreamer project
  + Transpilation of Faust to Python
  + AD via JAX

* The Faust Programming Language

Faust is a domain-specific, functional programming language for
digital signal processing and audio synthesis.

- Syntax describes a /circuit/ through which /signals/ propagate
- (Almost) everything in Faust is a signal, or something that operates on signals

** A basic Faust program

- Two input signals — implicit
- One output signal: the sum of the inputs

** A recursive Faust program

- One input
- One output: the sum of the input and scaled, delayed copy of the output

#+begin_src faust :results html :exports both :tab svg :sizes 0,100
process = +~(@(10000) : *(.5));
#+end_src
