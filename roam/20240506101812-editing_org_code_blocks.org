#+title: Editing Org code blocks

With cursor in a source block, hit =C-c ' (org-edit-special)= to edit
the code in a separate buffer. How lovely.

Using =org-edit-special= on =org= syntax returns headlines, source
blocks, etc. comma-prepended so they won't disappear on export.

#+begin_src org
,* This is an org headline

,#+begin_src emacs-lisp
(message "This is some elisp")
,#+end_src
#+end_src
