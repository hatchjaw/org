#+title: Emacs packages

YouTube recommended me [[https://www.youtube.com/watch?v=rwKTc4MNmt8][this video]].

Interesting Emacs packages:

- [[https://github.com/Alexander-Miller/treemacs][Treemacs]] A tree layout file explorer.
- [[https://github.com/hlissner/emacs-solaire-mode][Solaire mode]] ...an aesthetic plugin designed to visually distinguish
  "real" buffers from "unreal" buffers.
- [[https://github.com/roman/golden-ratio.el][Golden ratio]] ...automatically [resizes] the windows you are working
  on.
- [[https://github.com/tumashu/vertico-posframe][Vertio posframe]] Stick the vertico minibuffer in a modal overlay.
- [[https://github.com/seagle0128/doom-modeline][Doom modeline]] A sick modeline
- [[https://github.com/doomemacs/themes][Doom themes]] Sick themes
- [[https://elpa.gnu.org/packages/spacious-padding.html][Spacious padding]]
- [[https://github.com/domtronn/all-the-icons.el][All the icons]] Might not want this since I'm using nerd-icons already
  (which is more actively-maintained).
- [[https://github.com/emacs-dashboard/emacs-dashboard][Emacs Dashboard]] "An extensible emacs startup screen showing you
  what's most important."

For a rainy day, I suppose.
