* Fixed delay

Let's consider the case of delaying a function \(f(v)\), \(v = v(x)\),
by some fixed time \(\tau\):

#+NAME: eq:fixed-delay
\begin{align}
y = f(v) &= v(t - \tau).
\end{align}

The derivative of the above, via the [[denote:20240819T095042][Chain Rule]], is:

#+NAME: eq:diff-delay-1
\begin{equation}
\frac{dy}{dx} = \frac{dy}{dv}\frac{dv}{dx} \\
\end{equation}

But what is \(\frac{dy}{dv}\), i.e. the derivative of a delayed
signal?  First, the impulse response of the ideal delay system is a
time-shifted impulse:

#+NAME: eq:ideal-delay-ir
\begin{equation}
h(t) = \delta(t - \tau),
\end{equation}

and we can represent eqref:eq:fixed-delay as a convolution of \(v(t)\)
and that impulse response:

#+NAME: eq:conv
\begin{equation}
v(t - \tau) = v(t) \ast h(t).
\end{equation}

Next, recall the convolution property of the Fourier transform:

#+NAME: eq:fourier-conv
\begin{equation}
x(t) \ast h(t) \quad\xleftrightarrow{\mathcal{F}}\quad X(j\omega)H(j\omega),
\end{equation}

which we could also notate as:

#+NAME: eq:conv-mult
\begin{equation}
\mathcal{F}[x(t) \ast h(t)] = \mathcal{F}[x(t)]\cdot\mathcal{F}[h(t)] = X(j\omega)H(j\omega),
\end{equation}

and the [[denote:20240819T103529][derivative property]]:

#+NAME: eq:fourier-diff
\begin{equation}
\frac{d}{dt}x(t) \quad\xleftrightarrow{\mathcal{F}}\quad j\omega{}X(j\omega),
\end{equation}

from which it can be deduced that:

#+NAME: eq:fourier-conv-diff
\begin{equation}
  \begin{rcases}
    \frac{d}{dt}(x(t) \ast h(t)) \\
    \frac{d}{dt}(x(t)) \ast h(t) \\
    x(t) \ast \frac{d}{dt}(h(t))
  \end{rcases} \quad\xleftrightarrow{\mathcal{F}}\quad j\omega{}X(j\omega)H(j\omega).
\end{equation}

To put it another way, using the notation from eqref:eq:conv-mult:

#+NAME: eq:diff-conv
\begin{align}
\mathcal{F}\left[\frac{d}{dt}(x(t) \ast h(t))\right]
  &= \mathcal{F}\left[\left(\frac{d}{dt}x(t)\right)\right]\cdot\mathcal{F}[h(t)] \\
  &= \mathcal{F}[x(t)]\cdot\mathcal{F}\left[\left(\frac{d}{dt}h(t)\right)\right] \\
  &= j\omega{}X(j\omega)H(j\omega)
\end{align}

Anyway, thus:

#+NAME: eq:diff-delay
\begin{align}
\frac{dy}{dx} &= \frac{dv}{dx}\frac{d}{dv}(v(t) \ast h(t)) \\
  &= \frac{dv}{dx}\frac{d}{dv}(v(t)) \ast h(t) \\
  &= \frac{dv}{dx} \ast h(t) \\
  &= v'(t - \tau).
\end{align}

Or as a dual number, in discrete time:

#+NAME: eq:diff-delay-dual
\begin{equation}
\langle v,v' \rangle[n - k] = \langle v[n - k], v'[n - k] \rangle
\end{equation}

And in Faust:

#+begin_src faust :results none
diff(mem) = mem,par(i, N, mem);
#+end_src

* Variable Delay

This is where things get a little spicy. Consider a function of the
following form:

#+NAME: eq:variable-delay
\begin{equation}
y = f(\mathbf{v}) = v_{1}(t - v_{2}(t)), \quad v_{j}(\mathbf{x}) = v_{j}(x_{1}, x_{2}).
\end{equation}

It's derivative, via the chain rule, with respect to \(x_{i}\) is:

#+NAME: eq:variable-delay-deriv
\begin{equation}
\frac{\partial{}y}{\partial{}x_{i}} = \frac{\partial{}y}{\partial{}v_{1}}\frac{\partial{}v_{1}}{\partial{}x_{i}} + \frac{\partial{}y}{\partial{}v_{2}}\frac{\partial{}v_{2}}{\partial{}x_{i}}.
\end{equation}

From eqref:eq:diff-delay we know that the first term reduces to
\(\frac{\partial{}v_{1}}{\partial{}x_{i}} \ast h(t) = v_{1}'(t - v_{2}(t))\). We have nothing in
particular to fear from \(\frac{\partial{}v_{2}}{\partial{}x_{i}} = v_{2}'\). Let's tackle the
remaining term:

#+NAME: eq:troublesome-term
\begin{equation}
\frac{\partial{}y}{\partial{}v_{2}} = \frac{\partial{}}{\partial{}v_{2}}v_{1}(t - v_{2}) = \frac{\partial{}}{\partial{}v_{2}}(v_{1}(t) \ast \delta(t - v_{2}(t))).
\end{equation}

Using the [[denote:20240819T103529][derivative]] and multiplication properties of the Fourier
transform, we know we only have to differentiate one of the functions
in the convolution. Unfortunately, it's the delta function, because
that's the only one with a dependence on \(v_{2}\). Nontheless, that
gives us:

#+NAME: eq:troublesome-2
\begin{equation}
\frac{\partial{}y}{\partial{}v_{2}} = v_{1}(t) \ast \frac{\partial{}}{\partial{}v_{2}}\delta(t - v_{2}(t)).
\end{equation}

To tackle /this/, we can use a symbolic integral: the convolution
integral. In the general case of convolution with an impulse:

#+NAME: eq:conv-integral
\begin{equation}
(f \ast \delta)(t) = \int^{\infty}_{-\infty}f(t)\delta(t)\,dt = f(0).
\end{equation}

In the case of convolution with the derivative of an impulse, we can
use integration by parts:

#+NAME: eq:conv-integral-diff
\begin{align}
(f \ast \delta')(t) &= \int^{\infty}_{-\infty}f(t)\delta'(t)\,dt \\
  &= f(t)\delta(t)\Big|^{\infty}_{-\infty} - \int^{\infty}_{-\infty}f'(t)\delta(t)\,dt.
\end{align}

The surface term vanishes because the delta function is only nonzero
for \(t = 0\), which leaves us with the following nifty definition:

#+NAME: eq:conv-with-diff-delta
\begin{equation}
(f \ast \delta')(t) = -(f' \ast \delta)(t).
\end{equation}

Introducing a time-shift:

#+NAME: eq:conv-with-diff-shift
\begin{align}
f(t) \ast \frac{d}{dk}\delta(t - k) &= -\frac{d}{dk}(f(t) \ast \delta(t - k)) \\
  &= -\frac{d}{dk}f(t - k).
\end{align}

(Should the right hand side be \(\frac{d}{dt}\)?)

Now we can build on eqref:eq:troublesome-2:

#+NAME: eq:troublesome-cont
\begin{align}
\frac{\partial{}y}{\partial{}v_{2}} &= v_{1}(t) \ast \frac{\partial{}}{\partial{}v_{2}}\delta(t - v_{2}(t)) \\
  &= -\frac{\partial{}}{\partial{}v_{2}}v_{1}(t - v_{2}(t)),
\end{align}

and the full expression for a differentiable, variable delay is:

#+NAME: eq:diff-variable-delay
\begin{align}
\frac{\partial{}y}{\partial{}x_{i}} &= \frac{\partial{}}{\partial{}x_{1}}v_{1}(t - v_{2}(t)) -
  \frac{\partial{}v_{2}(t)}{\partial{}x_{i}}\frac{\partial{}}{\partial{}v_{2}}v_{1}(t - v_{2}(t)) \\
  &= v_{1}'(t - v_{2}(t)) - v_{2}'(t)\frac{\partial{}}{\partial{}v_{2}}v_{1}(t - v_{2}(t)).
\end{align}

But this isn't nice... should that be \(\frac{\partial{}}{\partial{}t}v_{1}\)? That's what
we worked out last year, but my notes on the topic aren't convincing.

** Intuition

Let \(v_{1}(x_{1}, x_{2}) = \sin(x_{1}(t))\), and \(x_{1}(t) = 2\pi{}t\).

#+begin_src python :session :results none :exports none
import matplotlib.pyplot as plt
import numpy as np
plt.figure(figsize=(10,5))
plt.grid()
N = 1000
t = np.linspace(0, 1, N, endpoint=False)
v1 = np.tile(np.sin(2 * np.pi * t * 1), 2)
plt.close()
#+end_src

- If delay \(v_{2}\) stays constant, \(v_{1}\) is reproduced at original
  rate with fixed delay. E.g. constant delay of \(v_{2} = 0.1\):

#+begin_src python :results file :session :var matplot_lib_filename=(org-babel-temp-file "figure" ".png") :exports results
v2 = .1
k = int(v2 * N)
for n in np.arange(N):
    y[n] = v1[n - k] if n - k >= 0 else 0
plt.clf()
plt.plot(t, v1[0:N], t, y)
plt.tight_layout()
plt.savefig(matplot_lib_filename)
matplot_lib_filename
#+end_src
